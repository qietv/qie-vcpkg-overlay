# qie-vcpkg-overlay

The vcpkg overlay ports and triplets for Qie C/C++ projects.

## Support overlay triplets

### Windows (Dynamic Library) with Mile.Windows.UniCrt

- arm64-windows-unicrt
- x64-windows-unicrt
- x86-windows-unicrt

### Android API Level 21 (Dynamic Library)

- arm-android-21
- arm64-android-21
- x64-android-21
- x86-android-21

### Android API Level 21 (Static Library)

- arm-android-21-static
- arm64-android-21-static
- x64-android-21-static
- x86-android-21-static
