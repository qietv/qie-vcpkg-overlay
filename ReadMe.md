# qie-vcpkg-overlay

The vcpkg overlay ports and triplets for Qie C/C++ projects.

## Support overlay triplets

- arm-android-21
- arm64-android-21
- arm64-windows-unicrt
- x64-android-21
- x64-windows-unicrt
- x86-android-21
- x86-windows-unicrt
